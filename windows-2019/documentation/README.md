# CIS Documentation

Place the official CIS Microsoft Windows Server 2019 Benchmark PDF document in this directory.

## Download Location
The official CIS benchmark can be downloaded from:
https://www.cisecurity.org/benchmark/microsoft_windows_server

## File Naming
Recommended filename: `CIS_Microsoft_Windows_Server_2019_Benchmark_v1.x.x.pdf`

## Usage
When the scanner runs, it will automatically copy any PDF files from this directory to the reports folder for easy reference.
