# Windows Server 2019 - Known Scanner Bugs

## False Positives (Scanner reports FAIL but setting is correct)

### ALL SECTION 2.3.x CONTROLS - Registry Check Bug
**Issue**: All Local Policies (Section 2.3.x) are reporting FALSE POSITIVES
**Root Cause**: Registry path or value check logic is incorrect
**Workaround**: Manually verify all 2.3.x controls via Group Policy Editor and document as PASS with screenshots

**Affected Controls:**
