===============================================================
Scanning Methodology
===============================================================

This scanner performs 100% READ-ONLY audits:
  ✓ Reads security policies via secedit
  ✓ Reads audit policies via auditpol
  ✓ Reads registry settings
  ✓ Reads firewall configuration
  ✗ Does NOT make any changes
  ✗ Does NOT remediate findings
  ✗ Does NOT modify system settings

All findings must be reviewed by a qualified administrator
before any remediation is performed.

===============================================================
Output Formats
===============================================================

The scanner generates:
  - CSV: Machine-readable results
  - HTML: Interactive web report
  - PDF: Printable report (via browser)
  - Word: Editable document

Each report includes:
  - Control ID and title
  - Current status (PASS/FAIL/N/A)
  - Actual value found
  - Evidence command used
  - Remediation guidance
  - CIS reference link
